# Hamming XOR Communication System

A C-based simulation of secure data transmission using:

- XOR Encryption
- Hamming (7,4) Error Correction
- Noisy Channel Simulation

## Features

- Converts text to binary
- Encrypts using XOR
- Encodes using Hamming(7,4)
- Simulates channel noise
- Detects and corrects single-bit errors
- Recovers original message
